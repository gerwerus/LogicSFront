# LogicS (Frontend)
Приложение для моделирования и расчёта логических схем
## Описание
Приложение позволяет создавать, сохранять, редактировать и просчитывать в реальном времени логические схемы
## Функционал
+ Роли с правами
+ + Администратор
+ + Учитель
+ + Ученик
+ Авторизация
+ Canvas для редактирования схем
## Запуск
```
npm i
npm run start
```
## Стек:
+ JS + React
